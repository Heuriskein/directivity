SETUP (Things to do before trying to run the program)
======================================================
1. Make sure you have Python 3.x installed.
2. Make sure you have pip installed
3. Install pipenv `pip install --user pipenv`
4. In the same directory as this file: `pipenv install`

RUNNING (Running the program)
======================================================
1. In the same directory as this file: `pipenv run rpd.python py`
